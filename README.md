# 🍊BITAmin — Scratch Implementations (Week-by-Week)

BITAmin 세션을 공부하면서 **딥러닝/머신러닝 핵심 모델을 구현**한 기록을 모아둔 repository입니다.  
주요 모델을 직접 코드로 구현하며 개념을 체득하는 것이 목표입니다.

📌 Winter Session 
✅ Week1: 퍼셉트론, 신경망 기본 구조 / 오차역전파, 경사하강법 
✅ Week2: 배치정규화, 최적화 기법 / CNN, Regularization 
✅ Week3: AutoEncoder, GAN, Diffusion / RNN, LSTM, GRU 
✅ Week4: Transformer, BERT, GPT / ViT, Swin Transformer 
✅ Week5: LLM, VLM / Fine Tuning, LoRA, Distillation
